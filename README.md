# While Loop

* while loop - repeated if statement

while condition:
  expression
1. Numerically calculating model
2. "Repeating action until the condition is met"

Ex. 
Error starts at 50
Divide error by 4 on every run
Continue until error no longer > 1

error = 50

while error > 1:
  error = error / 4
  print(error)
  
---> 12.5
---> 3.125
---> 0.78125
